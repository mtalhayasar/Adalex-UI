Features

  • Touch/swipe support for mobile devices
  • Keyboard navigation (Arrow Left/Right, Space for play/pause)
  • Multiple variants: default, hero, card, thumbnail
  • Autoplay with customizable interval
  • Loop option for continuous navigation
  • Multiple slides per view option
  • ARIA labels for accessibility
  • Pause autoplay on hover
  • Smooth transitions with CSS animations
  • Responsive design for all screen sizes
  • HTMX compatible with idempotent initialization
  • Custom events: carousel:change, carousel:slide-start, carousel:slide-end
