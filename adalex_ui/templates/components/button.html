{# Button Component #} {# Parameters: - text (required): The button text/label - type (optional): Button type (button/submit), defaults to 'button' - variant (optional): Visual variant (primary/secondary/ghost/link), defaults to 'primary' - size (optional): Button size (sm/md/lg), defaults to 'md' - disabled (optional): Boolean, disables the button - icon (optional): Icon HTML to prepend to text - full_width (optional): Boolean, makes button full width #}
